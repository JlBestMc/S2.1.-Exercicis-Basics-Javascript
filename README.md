## 🛠️ Getting Started

### 1️⃣ Clone this repository

```bash
git clone https://github.com/JlBestMc/S2.1.-Exercicis-Basics-Javascript.git
cd JavaScript-1
```

### 2️⃣ Install Dependencies

Make sure you have Node.js installed. Then install the packages:

```bash
npm install
```
